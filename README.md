## Getting Started
```bash
# Clone the repository
git clone https://github.com/ramansrivastava/ragflow-pro.git
cd ragflow-pro

# Install dependencies
pip install -r requirements.txt
npm install

# Start development servers
python -m uvicorn app.main:app --reload
npm run dev
```
